# Python Inventory Management System

This project demonstrates a simple command-line inventory management program written in Python.

Features include:

- View inventory items
- Add new items
- Update existing items
- Delete items
- Generate inventory reports

The program uses dictionary data structures to store inventory data and includes error handling for invalid user input.

Skills demonstrated:

- Python programming
- Data structures (dictionaries)
- File handling
- Input validation
- Modular program design
